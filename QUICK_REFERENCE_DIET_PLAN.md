# 🎯 Quick Reference - Diet Plan Generation

## Two Ways to Get Your Diet Plan

### 🏥 Method 1: Health Analysis
```
Fill Form → Click "Analyze My Health" → Scroll Down → See Diet Plan
```

### 📸 Method 2: Food Tracker
```
Upload Photos/Type Food → Click "Analyze My Food Intake" → Scroll Down → See Diet Plan
```

---

## When to Use Each

| Use Health Analysis When... | Use Food Tracker When... |
|----------------------------|--------------------------|
| Starting for the first time | Tracking daily meals |
| Want medical assessment | Want nutrition analysis |
| Need baseline plan | Need to adjust plan |
| Weekly check-ups | Daily monitoring |

---

## What You'll See

### After Health Analysis:
- ✓ Health Summary
- ✓ Risk Level (Low/Medium/High)
- ✓ Nutrition Needs Table
- ✓ Health Alerts
- ✓ **Diet Plan** ← Here!
- ✓ Nutrition Chart
- ✓ AI Recommendations

### After Food Tracker:
- ✓ Detected Foods
- ✓ Nutrition Summary
- ✓ Recommendations
- ✓ **Diet Plan** ← Here!

---

## Troubleshooting

### Diet Plan Not Showing?

**Quick Fixes:**
1. ✓ Backend running? → `python backend/app.py`
2. ✓ All fields filled?
3. ✓ Scrolled down?
4. ✓ Check console (F12)?
